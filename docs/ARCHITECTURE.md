## Adding New Components

### Add a New Infrastructure Module

1. Create `infrastructure/modules/new_module/resource.bicep`
2. Update `infrastructure/main.bicep` to include the module
3. Add to `pipelines/infrastructure-pipeline.yml`:
   ```yaml
   values:
     - all
     - foundry_connection
     - new_module  # â† Add here
   ```
4. Update `scripts/deploy_infrastructure.py` to handle the new module

### Add a New Agent

1. Create `agents/new-agent.yaml`
2. Add to `pipelines/agent-pipeline.yml`:
   ```yaml
   values:
     - simple-agent.yaml
     - new-agent.yaml  # â† Add here
   ```
3. Run the pipeline and select the new agent
